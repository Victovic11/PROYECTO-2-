¿Cómo funciona el Front-end?

La web está compuesta de muchos documentos que están conectados entre sí a través de enlaces. Cuando quieres entrar a alguna pagina web escribes la URL en la barra del navegador, eso quiere decir que estás solicitando que te muestren una página web.


En el siguiente paso tu computador verifica qué servidor de software está corriendo el sitio. Ahí el servidor evalúa qué tipo de petición es la que estás haciendo. Si entras a cualquier sitio web, obtiene una petición GET, así que debe hacer una conexión a la base de datos para obtener el listado de cursos.


Algunas veces no se requiere conexión a la base de datos. Por ejemplo, al entrar a algún sitio web (algunos en especifico) no hace falta, ya que cuando inicias sesión, se hace una petición POST que se conecta a la base de datos para validar los accesos y el tipo de cuenta o suscripción que tienes registrada. Con esto, el Back-end devuelve la respuesta al servidor para que la empuje al navegador.


Finalmente, entra el Front-end, que recibe la información que le pasó el Back-end y la dispone en la interfaz del sitio, lo que en este caso sería el perfil del usuario. Esto es lo que finalmente se muestra en el navegador.
